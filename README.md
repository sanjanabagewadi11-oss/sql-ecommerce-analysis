# SQL E-commerce Analytics Project

## Objective
Analyze e-commerce data using SQL to identify revenue trends,
customer lifetime value (CLV), and churn.

## Tools
- SQL (MySQL)
- GitHub

## Analysis Performed
- Revenue analysis
- Customer Lifetime Value (CLV)
- Churn analysis using last order date

## Folder Structure
- sql/: SQL queries
- README.md: Project overview

## Key Skills
- Joins
- CTEs
- Aggregations
- Date-based analysis
